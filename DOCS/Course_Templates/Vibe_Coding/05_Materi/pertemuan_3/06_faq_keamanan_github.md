**Pertanyaan:**
_"Mas, saya kan menyimpan semua berkas kode website saya di GitHub yang ada di internet. Apakah kode rahasia saya di sana aman? Bagaimana jika ada orang jahat yang mencuri atau menyalin hasil kerja keras saya tanpa izin?"_

**Jawaban:**
_"Jawabannya adalah: **SANGAT AMAN!** Anda tidak perlu khawatir tentang privasi hasil karya Anda._

*Mari kita bedah sistem keamanan GitHub menggunakan analogi yang sangat sederhana:*

Bayangkan GitHub adalah sebuah **Gedung Brankas Raksasa** milik bank internasional terpercaya. Di dalam gedung tersebut, ada dua jenis kamar brankas yang bisa Anda sewa:

---

### 1. Kamar Brankas Umum (Public Repository)

Jika Anda menaruh proyek Anda di sini, brankas Anda memiliki pintu kaca transparan. Siapa pun yang lewat di koridor gedung bisa melihat isi kode Anda, menyalinnya, atau menjadikannya inspirasi. 

Namun tenang saja, mereka **tidak memiliki kunci untuk mengubah atau merusak** isi brankas Anda. Hanya Anda pemegang kunci sah yang bisa menambah atau mengurangi barang di dalamnya.

- *Kapan kita pakai ini?* Biasanya untuk memamerkan portofolio lamaran kerja, membagikan template gratis kepada dunia, atau berkontribusi pada proyek sosial terbuka (*Open Source*).

---

### 2. Kamar Brankas Pribadi (Private Repository)

Jika Anda menaruh proyek Anda di kamar ini, pintu brankas terbuat dari baja tebal tanpa kaca. Tidak ada satu pun orang di dunia ini yang tahu apa isinya, apalagi menyalinnya. Hanya Anda pemilik akun yang sah (dan orang-orang khusus yang Anda beri izin secara tertulis) yang bisa membuka dan melihat isi kodenya.

- *Kapan kita pakai ini?* Biasanya untuk membangun produk bisnis komersial rahasia, proyek klien berbayar, atau jika Anda belum percaya diri membagikan kode Anda ke publik.

Di kelas OverCode Academy, secara bawaan kita bisa memilih jenis **Private Repository** (Repositori Pribadi) saat membuat wadah proyek di GitHub. Hal ini memastikan bahwa kode aplikasi Anda aman terlindungi dari pandangan publik.

Selain itu, GitHub adalah platform kontrol versi terbesar di dunia yang dimiliki oleh Microsoft. Mereka menjaga keamanan kode jutaan perusahaan raksasa dunia lainnya. Jadi, Anda bisa tidur dengan nyenyak tanpa perlu takut kode website Anda akan dicuri orang!"
