Pada pertemuan kedua, kita sudah mulai bersenang-senang menghias website pertama kita menggunakan React dan Tailwind CSS. Kita sudah melihat warna-warni indah bermunculan di layar browser kita.

- *"Bagaimana kalau saya salah pencet, lalu kodenya rusak dan websitenya mati total?"*
