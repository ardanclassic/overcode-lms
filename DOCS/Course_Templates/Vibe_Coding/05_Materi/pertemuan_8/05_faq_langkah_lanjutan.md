# Script Materi: Pertemuan 8 - 05. FAQ Langkah Lanjutan setelah Lulus

## ❓ FAQ (Sering Ditanyakan): Langkah Lanjutan setelah Lulus

Selamat atas kelulusan Anda! Website Anda kini sudah online dan berjalan lancar. Namun, perjalanan belajar seorang kreator teknologi tidak pernah benar-benar berhenti. Berikut adalah kumpulan pertanyaan mengenai langkah pengembangan diri selanjutnya setelah Anda menyelesaikan program belajar ini.

---

### Q1: "Saya sangat menikmati proses membuat website di kelas ini. Apa langkah pertama yang sebaiknya saya lakukan jika ingin membuat proyek website baru sendiri?"

**Jawab:**
Langkah pertama yang paling direkomendasikan adalah **memodifikasi proyek akhir Anda saat ini**.

- Jangan langsung mencoba membuat aplikasi skala raksasa seperti kloningan Gojek atau Tokopedia (itu membutuhkan tim puluhan programmer profesional).
- Mulailah dengan membuat variasi aplikasi sederhana lain yang relevan dengan kehidupan sehari-hari Anda: misalnya membuat website buku tamu pernikahan digital, website katalog menu makanan restoran UMKM milik teman Anda, atau aplikasi pelacak keuangan pribadi.
- Gunakan folder baru, buka Antigravity IDE, dan mulailah berdiskusi dengan asisten AI menggunakan formula prompt yang sudah kita pelajari untuk merancang aplikasinya dari nol.

---

### Q2: "Jika saya ingin memperdalam ilmu coding saya agar bisa mengetik kode secara manual tanpa bantuan AI sepenuhnya, dari mana saya harus mulai?"

**Jawab:**
Jika Anda tertarik untuk mendalami jalur profesional sebagai developer perangkat lunak murni (_software engineer_), Anda bisa mulai mempelajari dasar-dasar bahasa pemrogramannya secara berurutan:

1. **HTML & CSS**: Pelajari konsep dasar penyusunan elemen halaman web dan pewarnaan layout.
2. **JavaScript**: Ini adalah bahasa wajib untuk interaktivitas web. Pelajari konsep variabel, fungsi (_function_), percabangan (_if-else_), dan perulangan (_looping_).
3. **React & TypeScript**: Masuk ke framework modern yang kita gunakan di kelas ini untuk memahami bagaimana komponen dirender secara efisien di belakang layar.
   Ada banyak kelas belajar gratis berkualitas tinggi di internet, seperti _freeCodeCamp_, _W3Schools_, atau saluran edukasi pemrograman di YouTube.

---

### Q3: "Apakah ada bahayanya jika saya selamanya mengandalkan AI (Vibe Coding) untuk membuat website dan tidak pernah belajar coding manual?"

**Jawab:**
Sama sekali tidak berbahaya untuk skala proyek pribadi, portofolio, atau aplikasi startup tahap awal (_prototype_). Vibe Coding dibentuk untuk membebaskan ide Anda tanpa hambatan teknis.

Namun, Anda harus menyadari batasannya:

- Jika Anda ingin membangun sistem aplikasi perbankan raksasa dengan tingkat keamanan tinggi atau memproses miliaran data secara sangat efisien, Anda tidak bisa hanya mengandalkan Vibe Coding acak. Anda membutuhkan pemahaman arsitektur sistem komputer yang mendalam untuk memvalidasi setiap baris kode buatan AI agar bebas dari celah kebocoran keamanan.
- Singkatnya: Vibe Coding sangat sempurna untuk **kecepatan berkreasi dan validasi ide**, sedangkan pemahaman coding manual sangat krusial jika Anda ingin berkarier profesional sebagai insinyur perangkat lunak inti di perusahaan teknologi besar.

---

### Q4: "Bagaimana cara saya membagikan website buatan saya agar mendapatkan pengguna asli dan tidak hanya dikunjungi teman sekelas?"

**Jawab:**

- **Hubungkan Domain Kustom**: Secara bawaan, Vercel memberikan alamat tautan gratis seperti `nama-proyek.vercel.app`. Anda bisa membeli alamat web kustom berbayar yang murah (seperti `.com`, `.net`, atau `.id` yang berharga sekitar 50 ribu hingga 150 ribu rupiah per tahun) dan memasangnya di dashboard Vercel Anda hanya dengan beberapa klik. Ini akan membuat website Anda terlihat sangat profesional dan dipercaya publik.
- **Optimasi SEO**: Mintalah asisten AI Anda untuk mengoptimasi Meta Tags, deskripsi gambar, dan struktur heading halaman web Anda agar website mudah ditemukan di halaman pertama pencarian Google saat orang mengetikkan kata kunci tertentu.
